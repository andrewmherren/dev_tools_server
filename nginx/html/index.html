📖 Quick Start

Indexing Documentation: Navigate to Docs MCP and click "Queue New Scrape Job" to index any documentation site.

MCP Integration: All containers have access to these services via the dev-network. Configuration is in .vscode/mcp.json.

Adding Services: Update docker-compose.yml and nginx/nginx.conf to add new tool UIs.

🔧 Service Endpoints

Docs MCP SSE: http://docs-mcp:6280/sse (internal) or http://localhost/docs-mcp/sse (host)

Ollama API: http://ollama:11434 (internal) or http://localhost/ollama (host)

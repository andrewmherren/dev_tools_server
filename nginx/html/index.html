📖 About This Infrastructure

This stack provides shared AI services across all projects on this machine. Each service runs in an isolated container with security hardening.

MCP Servers: docs-mcp and sonarqube expose MCP interfaces. Configure connections in your project's .vscode/settings.json.

Network Access: Services communicate via internal-network (isolated) and dev-network (project-accessible).

🔧 Service Endpoints for Projects

Docs MCP SSE: http://docs-mcp:6280/sse (via Docker network) or http://localhost/docs-mcp/sse (from host)

Ollama API: http://ollama:11434 (via Docker network) or http://localhost/ollama (from host)

SonarQube Web: http://localhost/sonarqube/ (from host)

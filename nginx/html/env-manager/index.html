Usage Instructions

Configure Copilot

Add to your .vscode/mcp.json: 

{
  "servers": {
    "env-manager": {
      "command": "docker",
      "args": ["exec", "-i", "env-manager", "node", "dist/stdio.js"]
    }
  }
}

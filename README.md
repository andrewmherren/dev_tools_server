# Secure Agentic Dev Environment (Balanced)

A reproducible, VS Code/Copilot-centric Docker environment for local AI-assisted development, background coding agents, and sandboxed QA/testing workflows.

## What it is for
- Running local AI/dev services (MCP, model gateway, analysis tools) in containers.
- Supporting human + agent workflows using worktrees.
- Reducing host risk while keeping day-to-day developer UX practical.

## Key guarantees (balanced profile)
- Cross-platform operation: Windows, macOS, Linux.
- Agent execution in containerized sandboxes.
- No default autonomous remote git push path.
- Reproducible startup via documented compose workflow.

## How to use
- Follow `QUICKSTART.md` for setup and daily commands.
- Use `PLAN.md` for roadmap, security posture, and phased implementation details.
